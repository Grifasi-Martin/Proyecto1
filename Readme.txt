El proyecto esta vinculado a MarketCons, un portal de compras de materiales para la construccion.

Estructura del proyecto:
La carpeta Proyecto1 contiene la configuración principal del proyecto, que no será utilizada en esta ocasión.
La carpeta MarketCons contiene la lógica de la aplicación web.
La carpeta templates contiene las plantillas HTML utilizadas para las páginas.

Orden recomendado para probar las funcionalidades ingresando a la página http://127.0.0.1:8000/MarketCons/inicio/
1- Realizar el registro de usuario completando todos los campos, la contrasena debe ser alfabetica 
2- Puede consultar en el sistema sus datos registrados para controlar si son correctos
3- Explore las diferentes secciones del sitio que se muestran en el navbar: Clientes, Productos, Centro de ayuda y Empleados. Si asi lo desea puede volver con el boton de inicio en el final de la pagina para explorar cada una de las secciones.
